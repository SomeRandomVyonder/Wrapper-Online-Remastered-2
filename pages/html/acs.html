Here's what you'll need to use this page:

  1. Windows ME or Windows XP operating system
  2. Internet Explorer version 6.0 or higher
  3. 200 MHz PC (or faster)
  4. At least 128 MB of RAM
  5. At least 661K free disk space for the Microsoft.
  6. You must have minimum of 10GB free space Agent components, 3.1 MB for Peedy and an additional 1.34 MB for the L&H TruVoice Text-To-Speech engine
  7. Windows 2000 users may also need to install the SAPI 4.0 runtime
  8. A Windows-compatible sound card
  9. A compatible set of speakers and additional microphone (recommended)
  10. You must to be Internet Explorer instead.
  11. Use IETab or Internet Explorer. This site will not work in certain browsers without an extension: Firefox, Opera, Edge, Chrome, Safari.
